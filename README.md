# Finance Bot for Java 💰
Telegram-бот для управления личными финансами, построенный на Spring Boot. Позволяет отслеживать расходы и доходы, управлять категориями и получать финансовую статистику прямо в Telegram.

# Используемые технологии 🛠️
* Java 21
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Maven
* TelegramBots API
* Docker
